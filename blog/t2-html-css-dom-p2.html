What is grid based design?

Grid based design is a design of webpage with layout using pre-made grid classes. '.row' class is just one row of 100% of width of the page, and there are 12 '.column' classes with width of 1/12 to 12/12. 12 columns can useful to divide the page into 2, 3, or 4 which is very handy when making layout with different sized boxes.

Whats all the hype about responsive webpages?

When a webpage is responsive, the contents in the page changes size depend on the screen size. Hence, the website can be opened with different kinds of devices like tablet PC or smart phone without any problem.

What is semantic structure?

HTML has many semantic tags such as header, paragraph, division, etc. and they provide information about the each content on the page. For example, h1, h2, h3 are all heading tags but h1 is larger heading than h2 and h2 is larger heading than h3. The web browser understands it and shows different font size automatically. Semantic tags are communication between the code and the website.
