What a responsive site is, and why responsiveness is important.

A responsive site means the contents in the webpage are flexible in size depending on the screen size. contents like texts and images become smaller in small screen and become bigger in large screen. It is important to have responsive site because these days, people access internet with many different electronic devices from desktop to mobile phone.

What mobile first design is, and why it's important.

Mobile first design is designing of mobile screen webpage first, and then larger screen sizes. Now people use their smart phone more often to access website. Also mobile phone screen is the smallest screen so it is easy to design to show contents and they can go into certain layout for bigger screens.

What frameworks are, and their pros and cons.

Framework is a basic structure of the website that is already made and can be provided for any other designs. It makes building the website easier and faster than doing it from the scratch. However, using the framework without understanding it might not help improving the programming skills.

What a wireframe is and why we use it.

Wireframe is used when planning for webpage layout. It can be simple drawing of the boxes of different contents. We need wireframe to explain and represent the design before we make it.
