Javascript Language

parentheses ( )

- e.g. '1. Execute a function var result = doSomething()' The parentheses in the examples are used when calling function. Function is some peace of codes which is defined to do small task which can be executed as many times. It looks like this :
function functionName(variable) { code }
Some variables that is needed to run functions go in the parentheses. If no variable is needed, leave it empty. When calling the function, required variable needs go inside.
functionName(variable)

brackets [ ]

Brackets usually defines arrays.
array = [a, b, c]
Array is a list which contains anything such as string, integer, and even other arrays. In order to access array, index number is used starting from 0, like this :
variable = array[index];

braces { }

Braces are used for Function or Objects. For the Function, function code is inside braces.
function f1() { code }

And for the Object, object properties are inside.
var object = {"prop1":"a","prop2":"b" }
Object is a list of properties. It differs from arry because object must contain PropertyName:property.

single quotes ' ' & double quotes " "

The single quotes and double quotes are used for strings.
var string = "Hello"
If quotes are not used, computer doesn't understand it as a string, hence it will make error.
