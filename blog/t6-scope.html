Scope

What 'scope' is, and how it works in JavaScript.

Scope in JavaScript shows that a variable is visible in which part of the code. There are two types: global and local. If a variable is first defined inside a function, it has a local scope, and will not be able to call or get from outside of the function. On the other hand, if a variable is defined outside of the function, it's a global scope and it can be used at outside and inside of the function.
