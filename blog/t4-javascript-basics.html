Javascript Basics

How does JavaScript compare to HTML and CSS?

HTML works on structural part of the webpage and CSS works on visual design. Javascript makes interactive elements on them using different functions.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

The control flows every line in the code. If there is loops (while loop, for loop, etc) the control executes the lines inside the loop over and over until the described condition is not matching. For example,
while (it is weekdays) {
I wake up at 7 and go to school; }
-> the while loop condition is weekdays, so I repeatedly go to school from Monday to Friday. And on Saturday, because the condition is not matching, I stop getting up early.

Explain the difference between accessing data from arrays and objects.

Arrays and Objects have similar structures, but different usages and code. Array is like a list that can contain anything, and Object contains Properties.
To access data from Arrays:
ArrayName[index];
To access data from Objects:
Objectname. property;
or, Objectname["property"];
-> Index number is required to access data in arrays, and PropertyName is required to acess data in objects.

Explain what functions are and why they are useful.

Functions are blocks of code which can be called in order to execute certain tasks.
function functionName(variable) { }
Function looks like this, and the code that will be run goes inside the curly bracket. We dont need to write same code over and over but it can be done easily by calling the function.
